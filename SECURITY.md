# Security

## Reporting Issues

Found a security problem? Email [iamhimateja@gmail.com](mailto:iamhimateja@gmail.com) directly.

Don't open a public issue for security bugs.

Include:
- What the problem is
- How to reproduce it
- Impact (if known)

I'll respond within a few days and work on a fix.

## Supported Versions

Only the latest release gets security fixes.

## Privacy

This app doesn't collect any data. Everything stays on your device:
- Music files are stored locally (IndexedDB or filesystem)
- Settings are stored locally
- No analytics, no tracking, no accounts
- Works offline

The code is open source - you can verify this yourself.

## For Contributors

- Don't commit secrets or API keys
- Keep dependencies updated
- Run `bun run lint` before commits
